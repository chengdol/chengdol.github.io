iTerm2 Tips

  1. imgcat script, presenting image in terminal:
    https://iterm2.com/documentation-images.html

  2. Hotkeys, the floating half-transparent terminal window:
    Preferences -> Keys -> Hotkey -> Create a Dedicated Hotkey Window.
    My custom hotkey is ctrl + shift + t

  3. locate the cursor: command + /

  4. send commands to all tabs and all sessions shift + command + i, abort use the same command.

Install ohmyz on Mac:

Then install Powerlevel10k theme, this theme is what exactly I need:

By default it uses robbyrussell theme, you can see it from ~/.zshrc file, the theme files are located in ~/.oh-my-zsh/themes folder

https://github.com/ohmyzsh/ohmyzsh/wiki/Themes

Next when you start a new terminal session, the Powerlevel10 configure wizard will be launched to set your prompt, is will automatically check and install the font for you. When select encoding, choose Unicode, otherwise no icon will show.
